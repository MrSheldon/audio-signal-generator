## Parts 
- Arduino UNO
- Breadboard (generic)
- Rotary potentiometer (generic)
- Trimmer
- Buzzer
- Resistor 221 ohm
- Resistor 330 ohm
- Jumper wires (generic)

## Notes
- Connect + (plus) and - (minus) rows on the breadboard to the 5V and GND pins on the arduino board 
- The trimmer controls the LCD intensity
